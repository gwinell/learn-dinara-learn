## Настройка clang-format

### Копирование конфигурации

Файл `.clang-format` уже лежит в `materials/linters/`. Скопируйте его в `src/`:

```bash
cp materials/linters/.clang-format src/
```

### Что такое clang-format

clang-format автоматически форматирует код по определённым правилам. Конфигурация School 21:

- **Стиль**: Google C++ Style Guide
- **Отступ**: 4 пробела
- **Длина строки**: максимум 110 символов

### Проверка стиля

```bash
clang-format -n src/hello.c
```

- Если нет ошибок — команда молча вернёт 0
- Если есть — покажет, что нужно изменить (строка:столбец → что должно быть)

### Автоформатирование в VS Code

1. Откройте настройки (Cmd+,)
2. Найдите `Editor: Format On Save` → включите
3. Найдите `C_Cpp: Clang_format_style` → установите `Google`

Теперь при Cmd+S файл автоматически форматируется.

### Горячие клавиши clang-format

| Действие | Горячая клавиша | Команда в терминале |
|----------|----------------|---------------------|
| Форматировать файл | Shift+Option+F (VS Code) | `clang-format -i src/file.c` |
| Проверить стиль | — | `clang-format -n src/file.c` |
| Форматировать в терминале | — | `clang-format -i src/*.c` |

## Quest 1. Introducing (hello.c)

### Задание

Создать программу `src/hello.c`, которая выводит "Hello, AI!".

### Разбор решения

```c
// src/hello.c

#include <stdio.h>  // Подключаем библиотеку для ввода/вывода

int main(void) {
    printf("Hello, AI!");  // Вывод текста без переноса строки
    return 0;              // Возврат 0 = программа завершилась успешно
}
```

### Разбор построчно

| Строка | Что делает | Почему |
|--------|-----------|--------|
| `#include <stdio.h>` | Подключает стандартную библиотеку ввода/вывода | Без неё `printf` не работает |
| `int main(void)` | Точка входа в программу | Каждая C-программа начинается с `main` |
| `printf("Hello, AI!");` | Печатает текст в консоль | `printf` = print formatted |
| `return 0;` | Завершает программу с кодом 0 | 0 = успех, ненулевой = ошибка |

### Компиляция и запуск

```bash
# Компиляция: gcc -Wall -Werror -Wextra -o имя_программы исходник.c
gcc -Wall -Werror -Wextra -o hello src/hello.c

# Запуск
./hello

# Вывод: Hello, AI!
```

### Что значат флаги gcc

| Флаг | Значение |
|------|----------|
| `-Wall` | Включить все основные предупреждения (Warning All) |
| `-Werror` | Считать каждое предупреждение ошибкой (Warning → Error) |
| `-Wextra` | Включить дополнительные предупреждения |
| `-o hello` | Имя выходного файла (без этого будет `a.out`) |

### Проверка стиля

```bash

# Должно быть пусто (без ошибок)
```

## Quest 2. Not My Name (named_hello.c)

### Задание

Создать программу `src/named_hello.c`, которая считывает целое число `name` и выводит "Hello, <name>!".

### Пример

| Ввод | Вывод |
|------|-------|
| 123 | Hello, 123! |

### Разбор решения

```c
// src/named_hello.c

#include <stdio.h>

int main(void) {
    int name;                    // Объявляем переменную для хранения числа
    scanf("%d", &name);          // Считываем число с клавиатуры
    printf("Hello, %d!", name);  // Выводим приветствие с числом
    return 0;
}
```

### Разбор построчно

| Строка | Что делает | Почему |
|--------|-----------|--------|
| `int name;` | Создаёт целочисленную переменную | Нужно куда-то записать введённое число |
| `scanf("%d", &name);` | Считывает число с stdin | `%d` = формат для целого числа, `&` = адрес переменной |
| `printf("Hello, %d!", name);` | Печатает строку с подставленным числом | `%d` заменяется на значение `name` |

### Важно: `&` в scanf

`scanf` требует **адрес** переменной (not значение), поэтому пишем `&name`. Если забыть `&` — будет крах (segmentation fault).

### Компиляция и запуск

```bash
gcc -Wall -Werror -Wextra -o named_hello src/named_hello.c
echo 123 | ./named_hello
# Вывод: Hello, 123!
```

`echo 123 |` передаёт число 123 в stdin программы (вместо ручного ввода).

## Quest 3. Arithmetic Test (arithmetic.c)

### Задание

Создать программу `src/arithmetic.c`, которая считывает два целых числа и выводит сумму, разность, произведение и частное (целочисленное деление), разделённые пробелами.

### Примеры

| Ввод | Вывод |
|------|-------|
| 8 2 | 10 6 16 4 |
| 1 0 | 1 1 0 n/a |
| 3 2 | 5 1 6 1 |

### Разбор решения

```c
// src/arithmetic.c

#include <stdio.h>

int main(void) {
    int a, b;
    if (scanf("%d %d", &a, &b) != 2) {
        printf("n/a");
        return 0;
    }
    if (b == 0) {
        printf("%d %d %d n/a", a + b, a - b, a * b);
    } else {
        printf("%d %d %d %d", a + b, a - b, a * b, a / b);
    }
    return 0;
}
```

### Разбор построчно

| Строка | Что делает | Почему |
|--------|-----------|--------|
| `int a, b;` | Две переменные для чисел | Нужны два числа для операций |
| `scanf("%d %d", &a, &b)` | Считывает два числа через пробел | `%d %d` = два целых числа |
| `!= 2` | Проверяет, считалось ли ровно 2 числа | Если ввели текст вместо числа — ошибка |
| `b == 0` | Проверка деления на ноль | Деление на ноль = крах программы |
| `a + b`, `a - b`, `a * b`, `a / b` | Арифметические операции | `/` для целых чисел = целочисленное деление |

### Обработка ошибок

- Если ввели не число → `scanf` вернёт не 2 → выводим `n/a`
- Если второе число = 0 → вместо частного выводим `n/a`

### Компиляция и запуск

```bash
gcc -Wall -Werror -Wextra -o arithmetic src/arithmetic.c
echo "8 2" | ./arithmetic
# Вывод: 10 6 16 4

echo "1 0" | ./arithmetic
# Вывод: 1 1 0 n/a
```

## Quest 4. Who Is Max? (max.c)

### Задание

Создать программу `src/max.c`, которая считывает два целых числа и выводит большее. Реализовать поиск максимума отдельной функцией.

### Примеры

| Ввод | Вывод |
|------|-------|
| 3 2 | 3 |
| 12.3 10 | n/a |

### Разбор решения

```c
// src/max.c

#include <stdio.h>

int max(int a, int b) {
    if (a > b) {
        return a;
    }
    return b;
}

int main(void) {
    int a, b;
    if (scanf("%d %d", &a, &b) != 2) {
        printf("n/a");
        return 0;
    }
    printf("%d", max(a, b));
    return 0;
}
```

### Разбор построчно

| Строка | Что делает | Почему |
|--------|-----------|--------|
| `int max(int a, int b)` | Объявление функции max | Нужна отдельная функция (требование задания) |
| `if (a > b) return a;` | Если a больше — возвращаем a | Логика поиска максимума |
| `return b;` | Иначе возвращаем b | b >= a |
| `max(a, b)` | Вызов функции с двумя числами | Подставляет a и b в функцию |

### Компиляция и запуск

```bash
gcc -Wall -Werror -Wextra -o max src/max.c
echo "3 2" | ./max
# Вывод: 3

echo "12.3 10" | ./max
# Вывод: n/a
```

## Quest 5. Complex Calculations (important_function.c)

### Задание

Создать программу `src/important_function.c`, которая вычисляет:

```
y = 7e-3 × x⁴ + ((22.8 × x^(1/3) – 1e3) × x + 3) / (x² / 2) – x × (10 + x)^(2/x) – 1.01
```

Разрешено использовать `<math.h>`.

### Пример

| Ввод | Вывод |
|------|-------|
| 1 | -2070.4 |

### Разбор решения

```c
// src/important_function.c

#include <math.h>
#include <stdio.h>

int main(void) {
    double x;
    if (scanf("%lf", &x) != 1) {
        printf("n/a");
        return 0;
    }

    double term1 = 7e-3 * pow(x, 4);
    double term2 = ((22.8 * pow(x, 1.0/3.0) - 1e3) * x + 3) / (x * x / 2.0);
    double term3 = x * pow(10 + x, 2.0 / x);
    double y = term1 + term2 - term3 - 1.01;

    printf("%.1f", y);
    return 0;
}
```

### Разбор построчно

| Строка | Что делает | Почему |
|--------|-----------|--------|
| `#include <math.h>` | Подключает математические функции | Нужны `pow()` для возведения в степень |
| `double x;` | Вещественная переменная | x может быть дробным |
| `scanf("%lf", &x)` | Считывает дробное число | `%lf` = double float |
| `pow(base, exp)` | Возведение в степень | `pow(x, 4)` = x⁴ |
| `pow(x, 1.0/3.0)` | Кубический корень | x^(1/3) |
| `2.0/x` | Деление на x | Нужен `2.0` (не `2`) для вещественного деления |
| `printf("%.1f", y)` | Вывод с 1 знаком после точки | `.1f` = один знак после запятой |

### Важные нюансы

1. **`1.0/3.0` вместо `1/3`**: Целочисленное деление `1/3 = 0`! Нужно `1.0/3.0 = 0.333...`
2. **`2.0/x` вместо `2/x`**: Та же проблема — целочисленное деление
3. **`pow` возвращает `double`**: Поэтому все переменные — `double`
4. **Компиляция с `-lm`**: На Linux нужен флаг `-lm` для математики. На macOS обычно не нужен, но лучше добавить:

```bash
gcc -Wall -Werror -Wextra -o important_function src/important_function.c -lm
echo 1 | ./important_function
# Вывод: -2070.4
```

## Quest 6. Be Careful with the Floating (float_compare.c)

### Задание

Исправить программу `src/float_compare.c`, чтобы она выводила "OK!". Проблема — сравнение float/double с нулём через `==`.

### Проблема

```c
// Исправленный файл src/float_compare.c
#include <math.h>
#include <stdio.h>

double fun(double x) {
    // Какая-то вычислительная функция
    // (её НЕЛЬЗЯ менять — только сравнение)
    return (x * x - 4.0);
}

int main(void) {
    double res = fun(2.0);  // Должно дать ~0, но из-за погрешности = 4.44e-16
    double epsilon = 1e-6;  // Маленькое число для сравнения

    if (fabs(res) < epsilon) {  // Было: if (res == 0)
        printf("OK!");
    } else {
        printf("NOT OK!");
    }
    return 0;
}
```

### Почему `== 0` не работает

Компьютер хранит дробные числа с有限ной точностью. Результат вычислений может быть `0.00000000000000044` вместо ровно `0`. Сравнение `0.00000000000000044 == 0` → `false`.

### Решение: epsilon

Сравниваем не с нулём, а с маленьким числом (epsilon):

```c
if (fabs(res) < epsilon)  // fabs = абсолютное значение для double
```

### Компиляция и запуск

```bash
gcc -Wall -Werror -Wextra -o float_compare src/float_compare.c -lm
./float_compare
# Вывод: OK!
```

## Quest 7. Lock Picking (crack.c)

### Задание

Создать программу `src/crack.c`, которая считывает координаты точки (x, y) и проверяет, лежит ли она строго внутри круга x² + y² = 25.

### Примеры

| Ввод | Вывод |
|------|-------|
| 1.5 1.5 | GOTCHA |
| 5 0 | MISS |

### Разбор решения

```c
// src/crack.c

#include <math.h>
#include <stdio.h>

int main(void) {
    double x, y;
    if (scanf("%lf %lf", &x, &y) != 2) {
        printf("n/a");
        return 0;
    }

    if (x * x + y * y < 25) {  // Строго внутри круга
        printf("GOTCHA");
    } else {
        printf("MISS");
    }
    return 0;
}
```

### Разбор построчно

| Строка | Что делает | Почему |
|--------|-----------|--------|
| `double x, y;` | Две переменные для координат | Координаты могут быть дробными |
| `x * x + y * y < 25` | Уравнение круга: x²+y² < r² | r² = 25, r = 5 |
| `printf("GOTCHA")` | Точка внутри | Строго меньше 25 = строго внутри |
| `printf("MISS")` | Точка на границе или снаружи | 25 и больше = MISS |

### Геометрия

Круг с радиусом 5, центр в (0,0). Точка (x,y) внутри, если расстояние от центра < 5:

```
sqrt(x² + y²) < 5
```

Но `sqrt` не нужен — можно сравнить квадраты: `x² + y² < 25`.

### Компиляция и запуск

```bash
gcc -Wall -Werror -Wextra -o crack src/crack.c -lm
echo "1.5 1.5" | ./crack
# Вывод: GOTCHA

echo "5 0" | ./crack
# Вывод: MISS
```

## Шпаргалка: основы C

### Типы данных

| Тип | Что хранит | Пример | Формат в scanf/printf |
|-----|-----------|--------|----------------------|
| `int` | Целое число | `42`, `-7`, `0` | `%d` |
| `double` | Дробное число | `3.14`, `-0.001` | `%lf` (scanf), `%f` (printf) |
| `char` | Символ | `'A'`, `'0'` | `%c` |

### Операторы

| Оператор | Значение | Пример | Результат |
|----------|---------|--------|-----------|
| `+` | Сложение | `3 + 2` | `5` |
| `-` | Вычитание | `3 - 2` | `1` |
| `*` | Умножение | `3 * 2` | `6` |
| `/` | Деление | `7 / 2` | `3` (целочисленное) |
| `%` | Остаток | `7 % 2` | `1` |
| `==` | Равно | `3 == 3` | `1` (true) |
| `!=` | Не равно | `3 != 2` | `1` (true) |
| `>` | Больше | `3 > 2` | `1` (true) |
| `<` | Меньше | `3 < 2` | `0` (false) |
| `>=` | Больше или равно | `3 >= 3` | `1` (true) |
| `<=` | Меньше или равно | `3 <= 2` | `0` (false) |

### Математические функции (math.h)

| Функция | Значение | Пример |
|---------|---------|--------|
| `pow(x, y)` | x в степени y | `pow(2, 3)` = 8 |
| `sqrt(x)` | Квадратный корень | `sqrt(16)` = 4 |
| `fabs(x)` | Абсолютное значение | `fabs(-5)` = 5 |
| `ceil(x)` | Округление вверх | `ceil(2.1)` = 3 |
| `floor(x)` | Округление вниз | `floor(2.9)` = 2 |

### Форматы вывода printf

| Формат | Описание | Пример |
|--------|---------|--------|
| `%d` | Целое число | `printf("%d", 42)` → `42` |
| `%f` | Дробное (6 знаков) | `printf("%f", 3.14)` → `3.140000` |
| `%.1f` | 1 знак после точки | `printf("%.1f", 3.14)` → `3.1` |
| `%.6f` | 6 знаков после точки | `printf("%.6f", 3.14)` → `3.140000` |
| `%s` | Строка | `printf("%s", "hi")` → `hi` |
| `%c` | Символ | `printf("%c", 'A')` → `A` |

### Быстрая компиляция в терминале

```bash
# Компиляция + запуск одной командой
gcc -Wall -Werror -Wextra -o hello src/hello.c && ./hello

# Компиляция + запуск + автоматический тест
gcc -Wall -Werror -Wextra -o hello src/hello.c && echo "" | ./hello
```
